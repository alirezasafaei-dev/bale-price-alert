# راهنمای کاربر بات نواکس

## معرفی

بات نواکس (@novax_price_bot) یک ربات تلگرام برای دریافت قیمت‌های لحظه‌ای بازار ایران و تنظیم هشدار قیمت است.

## دستورات اصلی

### شروع کار با بات

برای شروع، در تلگرام به آدرس `@novax_price_bot` بروید و دستور زیر را ارسال کنید:

```
/start
```

بات با پیام خوش‌آمدگویی و لیست دستورات پاسخ می‌دهد.

### دریافت قیمت‌های لحظه‌ای

برای دریافت آخرین قیمت‌های بازار ایران:

```
/prices
```

این دستور قیمت‌های زیر را نمایش می‌دهد:
- دلار آزاد
- طلای ۱۸ عیار
- سکه امامی
- تتر (USDT)

همه قیمت‌ها به **تومان** و با زمان **تهران** نمایش داده می‌شوند.

### تنظیم هشدار قیمت

برای دریافت اعلان زمانی که قیمت یک دارایی به حد مشخصی رسید:

```
/alert <نماد> <قیمت_تومان> <above|below>
```

**مثال‌ها:**

```
/alert USD 170000 above
```
هشدار زمانی که دلار از ۱۷۰,۰۰۰ تومان بالاتر رود.

```
/alert GOLD 19000000 below
```
هشدار زمانی که طلای ۱۸ عیار از ۱۹,۰۰۰,۰۰۰ تومان پایین‌تر بیاید.

**نمادهای پشتیبانی شده:**
- `USD` - دلار آزاد
- `GOLD` - طلای ۱۸ عیار
- `SEKKEH` - سکه امامی
- `USDT` - تتر

### مشاهده هشدارهای فعال

برای دیدن لیست هشدارهای فعال خود:

```
/alerts
```

### حذف هشدار

برای حذف یک هشدار، ابتدا با `/alerts` شناسه هشدار را پیدا کنید، سپس:

```
/delete <id>
```

**مثال:**

```
/delete a1b2c3d4
```

### راهنما

برای دیدن راهنمای کامل:

```
/help
```

## نکات مهم

1. **بررسی خودکار**: بات هر ۱۰ دقیقه یک‌بار قیمت‌ها را بررسی می‌کند
2. **اعلان یک‌بار**: هر هشدار فقط یک‌بار اعلان داده می‌شود
3. **محدودیت تکرار**: برای جلوگیری از spam، اعلان‌های تکراری حداقل ۱ ساعت فاصله دارند
4. **منبع قیمت**: قیمت‌ها از سایت TGJU دریافت می‌شوند
5. **واحد قیمت**: همه قیمت‌ها به **تومان** هستند (نه ریال)

## مثال کامل استفاده

```
# شروع بات
/start

# دریافت قیمت فعلی
/prices

# تنظیم هشدار برای دلار
/alert USD 175000 above

# مشاهده هشدارهای فعال
/alerts

# حذف هشدار (با شناسه دریافتی از /alerts)
/delete abc123
```
